# AMC Portal Demo

> 이 프로젝트의 개발 컨벤션·기술스택·환경변수·보안 주의사항·확정된 결정은
> **레포에 두지 않고 `convention-mcp` MCP 서버로만 제공**합니다.
> (규칙 원문의 손쉬운 파일 복제를 막기 위함)

## 🛑 규칙 참조 의무 — 코드 작업 전 반드시 수행

이 레포에서 **코드를 작성·수정·리뷰·설계하기 전에**, 반드시 `convention-mcp` MCP 서버를
호출해 해당 작업과 관련된 규칙을 먼저 확인하라. 추측으로 진행하지 말 것.

- 전체 규칙이 필요하면 → `get_development_guide`
- 특정 주제만 필요하면 → `search_development_guide("<키워드>")`
  (예: `"JWT"`, `"환경변수"`, `"패키지 컨벤션"`, `"NL2SQL"`, `"프론트엔드"`)
- 어떤 주제가 있는지 훑으려면 → `list_guide_sections`

여기엔 기술 스택(임의 변경 금지), 백엔드 패키지 컨벤션, 환경변수 키, 보안 주의사항
(시크릿 노출 금지 등), 외부 NL2SQL API 사용법, 확정된 결정이 모두 들어 있다.

## 연결 확인

`convention-mcp` MCP 서버가 연결돼 있어야 위 tool 을 쓸 수 있다.
- 연결 설정: 레포 루트 [.mcp.json](.mcp.json)
- 서버가 안 떠 있거나 tool 이 안 보이면, 사용자에게 MCP 서버 기동/연결을 요청하라
  (서버 코드: `~/Projects/AMC-claude-MCP`).
- **tool 을 호출할 수 없는 상태에서 규칙을 임의로 가정해 작업하지 말 것.**
